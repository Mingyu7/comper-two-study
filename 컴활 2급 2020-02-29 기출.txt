멀티미디어의 특징
- 다양한 아날로그 데이터를 디지털 데이터로 변환하여 통합 처리함
- 정보 제공자의 선택에 의해 일방적으로 데이터가 전달되는 것이 아니라 정보 제공자와 사용자 간의 의견을 통한 상호작용에 의해 데이터가 전달됨
- 데이터가 일정한 방향으로 순차적으로 처리되는 것이 아니라 사용자의 선택에 따라 다양한 방향으로 처리됨
- 텍스트, 그래픽, 사운드, 동영상, 애니메이션 등의 여러 미디어를 통합 처리함

WAVE(웨이브)
-직접 재생이 가능
-자연음향, 사람음성 표현 가능
-음질이 뛰어나지만 파일의 용량이 크다.
-Windows Media Player로 파일 재생 가능
-확장자: *.WAV

데이터 보안 침해 형태중 하나인 변조 : 전송된 데이터를 다른 내용으로 바꾸는 것
인터넷의 표준 주소 체계인 URL ->  URL 표기방법 : 프로토콜://서버주소[:포트번호]/파일경로
가상 메모리 : 주기억장치의 크기보다 큰 용량을 필요로 하는 프로그램을 실행해야 할 때 유용하게 사용된다.
전자 우편은 7비트 ascii 코드를 사용한다
IoT 네트워크는 비용이 비싸고, 각기기간의 보안문제가 커진다
이미지를 얼마나 세밀하게 표시할 수 있는가를 나타내는 정보로 픽셀수에 따라 결정되는 것은 = 해상도
** ~이면서 = and , ~ 이거나 = or **

-다른 시트에 있어야 '원본데이터 연결' 가능

메크로
[옵션] 단추클릭하면 선택한 매크로의 매크로 설명을 수정할수있는 대화상자 표시 (이름 변경 불가!)

함수
=COUNTIF(조건범위, 조건)
MID는 시작위치
=MID(텍스트, 시작위치, 추출할 글자수)
LEFT 함수는 맨 앞
=LEFT(텍스트, 추출할 글자수)
RIGHT 함수는 맨 끝
=RIGHT(텍스트, 추출할 글자수)
=VLOOKUP(찾을값, 참조범위, 열번호, 옵션)

	0 또는 FALSE : 찾으려고 하는 값 (찾는값) 이 문자인 경우에 사용하고 정확하게
	일치하는 값을 찾을 때 사용합니다.
	
	생략 또는 TRUE : 찾으려고 하는 값 (찾는값) 이 숫자인 경우에 사용하고 (오름차순으로 정렬이 되어있어야 한다)
	찾으려는 값보다 더 작은값을 찾거나 비슷한 값, 근사값을 찾으려고 할 때 사용합니다.

페이지 나누기
 1.	페이지 나누기는 워크시트를 인쇄할 수 있도록 페이지 단위로 나누는 구분선이다.
     2.	[페이지 나누기 미리 보기] 상태에서 마우스로 페이지 나누기 구분선을 클릭하여 끌면 페이지를 나눌 위치를 조정할 수 있다.
     3.	행 높이와 열 너비를 변경해도 자동 페이지 나누기 구분선의 위치는 변경되지 않는다. !! ( 위치가 변경되어 페이지수 늘어남)
     4.	[페이지 나누기 미리 보기] 상태에서 파선은 자동 페이지 나누기를 나타내고 실선은 사용자 지정 페이지 나누기를 나타낸다.
